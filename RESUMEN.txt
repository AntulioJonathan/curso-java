🚀 CÓMO USAR
=============

OPCIÓN 1: Ejecutar un archivo específico
-----------------------------------------
cd 01_fundamentos
javac SintaxisBasica.java
java fundamentos.SintaxisBasica

OPCIÓN 2: Compilar todos en una carpeta
---------------------------------------
javac -d . 01_fundamentos/*.java
java fundamentos.SintaxisBasica

OPCIÓN 3: Usar script en Linux/Mac
----------------------------------
chmod +x ejecutar.sh
./ejecutar.sh

OPCIÓN 4: Usar script en Windows
-------------------------------
compilar.bat
java -cp bin fundamentos.SintaxisBasica

OPCIÓN 5: Proyecto integrador
-----------------------------
javac ProyectoIntegrador.java
java proyecto_integrador.ProyectoIntegrador

OPCIÓN 6: En VS Code/IntelliJ IDEA
---------------------------------
1. Abre el archivo
2. Presiona "Run" o Ctrl+F10
